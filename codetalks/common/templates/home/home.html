Introduction

Hello and welcome to the codebin project! This website is an online Python 2.7 development enviroment. This site is intended to be a resource for the quick development of small python scripts as well as larger projects. Don't expect any extreme performance because the Python is interpreted by a python interpreter known as Skulpt; which is written in Javascript!

Capabilities

This website has the capability to save your code, search for projects, rename projects, make projects public or private, and fork projects. You can keep track of projects by recording the permalinks or by searching for the projects with the built in search feature.
